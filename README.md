## Name

Translate

## Description

Translation manager.

## Installation

```
$> git clone https://gitlab.com/jbaenava00/translate.git
$> cd translate
$> git checkout 1.0.0
$> npm install
$> npm start
```

# Instructions to develop

```
$> git clone https://gitlab.com/jbaenava00/translate.git
$> cd translate
$> git checkout develop
$> npm install
```

## Start application for development

```
$> npm run dev

```
